# 营收3表数字列排序功能计划

## 摘要
给 RevenueSummary、RevenueDetail、RevenueDeviation 三个报表中所有数字列添加表头点击排序功能。

## 技术方案
Element Plus el-table 原生 `sortable="custom"` + `@sort-change`：
- 给数字列添加 `sortable="custom"` 属性
- el-table 监听 `@sort-change` 事件
- 新增 sortProp/sortOrder ref，在 filteredData computed 中排序后再分页

## 改动文件

### 1. RevenueSummary.vue

**施工类表（L119 el-table）**：添加 `@sort-change="handleSortChange"`
**数字列（30列）添加 `sortable="custom"`**：
- revenueTarget, completionRate
- carryForwardConstruction, completedPendingSettlement, carryForwardTotal
- monthlyPlanConstruction, monthlyPlanCompleted, monthlyPlanNew, monthlyPlanTotal
- monthlyActualConstruction, monthlyActualCompleted, monthlyActualNew, monthlyActualTotal
- planConstruction, planCompleted, planNew, planTotal
- actualConstruction, actualCompleted, actualNew, actualTotal
- reportedRevenue, newContractAmount, remainingContract

**产品类表（L486 el-table）**：添加 `@sort-change="handleProductSortChange"`
**数字列**：revenueTarget, completionRate, annualPlan, monthlyActual, annualActual, nextMonthPlan → 添加 `sortable="custom"`

**其他类表（L586 el-table）**：添加 `@sort-change="handleOtherSortChange"`
**数字列**：monthlyActual, annualActual → 添加 `sortable="custom"`

**脚本新增**：
```js
const sortProp = ref('')
const sortOrder = ref('')
const handleSortChange = ({ prop, order }) => {
  sortProp.value = prop || ''
  sortOrder.value = order || ''
}
```
在 `filteredConstructionData` computed 中，return 前加排序逻辑。

### 2. RevenueDetail.vue

**el-table（L131）**：添加 `@sort-change="handleSortChange"`
**数字列（13列）添加 `sortable="custom"`**：
- contractAmount, carryForwardRevenue, annualPlanRevenue, annualEstimatedRevenue
- planAdjustmentRate, monthActualRevenue, annualAccumulatedRevenue
- startAccumulatedRevenue, monthReportedRevenue, annualReportedRevenue
- totalReportedRevenue, remainingContractAmount, progress

**脚本新增**：sortProp/sortOrder ref + handleSortChange + 在 filteredData computed 排序

### 3. RevenueDeviation.vue

**当月偏差表（L98 el-table）**：添加 `@sort-change="handleMonthlySortChange"`
**数字列（11列）添加 `sortable="custom"`**：
- contractAmount, carryForwardRevenue, annualPlanRevenue, annualAccumulatedRevenue
- totalAccumulatedRevenue, monthPlanRevenue, monthActualRevenue, completionRate
- deviation, negativeDeviation（monthPlanRevenue/monthActualRevenue 无 prop 需补加）

**季度偏差表（L451 el-table）**：添加 `@sort-change="handleQuarterlySortChange"`
**数字列（8列）添加 `sortable="custom"`**：
- contractAmount, carryForwardRevenue, annualPlanRevenue, annualAccumulatedRevenue
- totalAccumulatedRevenue, quarterOriginalPlan, quarterRevisedPlan, planReductionAmount

**脚本新增**：4个 ref + 2个 handler + 在 filteredData/quarterlyFilteredData computed 中排序

## 验证
- 点击任意数字列表头 → 升序/降序切换
- 切换排序后分页数据正确
- 搜索/重置后排序状态清除
