# 当年累计完成营收情况悬浮窗去掉区域分部显示

## 概述
在产运看板（UnifiedDashboard）的"当年累计完成营收情况"图表中，tooltip 悬浮窗底部显示了"区域分布"明细（上海、长三角总部等6个区域的项目数量），用户要求去掉这部分内容。

## 修改位置

### 文件1：UnifiedDashboard.vue（主文件 - 产运看板）
**路径**: `e:\trae_demo_env\营收看板\src\views\UnifiedDashboard.vue`
**函数**: `initSectorChart()` 内的 tooltip.formatter
**位置**: 约 L944-966

**删除内容** — tooltip formatter 中 `// 显示全部6个区域的项目数量` 到对应的 `result += '</div>'` 之间的代码块：
```js
// 删除以下7行:
result += `<div style="margin-top:6px;padding-top:6px;border-top:1px dashed #eee;font-size:11px;color:#666;">`
result += `<div style="font-weight:bold;margin-bottom:3px;">区域分布:</div>`
allRegionSummary.forEach(r => {
  result += `<div style="display:flex;justify-content:space-between;gap:12px;"><span>${r.name}</span><b style="color:#1890ff;">${r.count}个</b></div>`
})
result += `<div style="margin-top:4px;padding-top:4px;border-top:1px solid #eee;display:flex;justify-content:space-between;"><span>合计</span><b style="color:#f5222d;">${totalRegionCount}个项目</b></div>`
result += `</div>`
```

**同步清理**: 可选删除 `allRegionSummary` 数组和 `totalRegionCount` 变量定义（约 L933-941），因为不再使用。

### 文件2：RevenueDashboard.vue（独立营收看板页面）
**路径**: `e:\trae_demo_env\营收看板\src\views\RevenueDashboard.vue`
**同样的修改** — tooltip formatter 中相同的区域分布代码块（约 L752-759），以及可选清理 allRegionSummary（约 L729-737）

## 验证方式
1. 在产运看板中打开"当年累计完成营收情况"图表
2. 鼠标悬浮柱子/折线点，tooltip 不再显示"区域分布"部分
3. 只保留基础数据（累计营收、项目数量等）
