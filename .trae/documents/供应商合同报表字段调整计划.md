### 步骤2：新增 getWarningLabel 函数 + 更新 getSignStatusClass
**文件**: [SupplierContractReport.vue](src/components/supply-chain/SupplierContractReport.vue#L942-L949)
- **在 getSignStatusClass 函数之后**添加新函数 `getWarningLabel(row)`：
  ```js
  const getWarningLabel = (row) => {
    if (row.signStatus === '已签订') return '已签订'
    const days = row.overdueDays
    if (days >= 60) return '红色预警'
    if (days >= 45) return '橙色预警'
    if (days >= 21) return '黄色预警'
    return '未签订'
  }
  ```
- **更新 getSignStatusClass**：为所有返回值追加 `font-medium` 类名（保持颜色一致）

### 步骤3：rawData 补充 contractStartDate / contractEndDate 字段
**文件**: [SupplierContractReport.vue](src/components/supply-chain/SupplierContractReport.vue#L341-L831)
- 为 rawData 中每个 contract 对象添加两个字段：
  - `contractStartDate`: 合同开始时间（基于 plannedDate 推算，通常比计划签订日期晚或相同）
  - `contractEndDate`: 合同计划/实际结束时间（基于项目周期推算）
- 共涉及约40个 contract 对象，分布在9个项目中
- 数据规则：
  - 已签订的合同：startDate ≈ actualDate 或稍后；endDate 设为合理未来日期
  - 未签订的合同：startDate 可留空或设为预计值；endDate 同理
