# 营收偏差情况按筛选栏过滤

## 修改
**文件**: [UnifiedDashboard.vue](src/views/UnifiedDashboard.vue)

### 问题
`unitDeviationData` 是静态数据（9家单位全部显示），不随顶部筛选栏变化。

### 方案
将 `unitDeviationData` 从 `ref` 改为 **computed**，根据 `selectedFilter` 过滤：

```js
// 改前
const unitDeviationData = ref([...9条全量数据...])

// 改后
const allUnitDeviationData = [...9条全量数据...]  // 原始数据移为常量

const unitDeviationData = computed(() => {
  if (selectedFilter.value === '全部') return allUnitDeviationData
  return allUnitDeviationData.filter(item => item.name === selectedFilter.value)
})
```

### 同步修改的依赖项
- `totalDeviationCount` — 已是 computed，基于 `unitDeviationData.value`，无需改动
- `getCircleOffset(i)` — 已基于 `unitDeviationData.value[i]`，无需改动
- 模板中 `v-for="(unit, idx) in unitDeviationData"` — 无需改动（自动响应）

### 效果
| 筛选选择 | 偏差情况显示 |
|---------|------------|
| 全部 | 9家单位 |
| 管网事业部 | 仅管网事业部1条 |
| 生态事业部 | 仅生态事业部1条 |
| ... | ... |
