# 供应链看板布局优化 + 供应商合同签订图例下移

## 概述
1. 修复供应链看板整体布局，让各组件铺满空白区域
2. 供应商合同签订(SupplyContractChart)的图例从顶部移到底部(ECharts内置)

---

## 当前布局分析（SupplyChainDashboard.vue）

```
┌─────────────────────────┬──────────────┐
│  Row 1: col-span-2      │ col-span-1    │
│  SupplyChainTimeliness  │ SupplyContract│
│  (圆环+文字，高度固定)    │ (柱状图 h-64) │
├─────────────────────────┼──────────────┤
│  Row 2: col-span-2      │ col-span-1    │
│  供应商统计(2个图表)     │ 材料集采       │
│  每个h-64               │ h-64          │
└─────────────────────────┴──────────────┘
```

**空白原因**：
- 各组件内部使用固定高度 `h-64`(256px)，不随容器拉伸
- 外层 grid 没有设置行高为 `auto`/`min-h`
- SupplyContractChart 有自定义图例在顶部（第21-26行），ECharts legend 为 `show: false`

---

## 修改方案

### 调整1：SupplyContractChart — 图例下移

**文件**: [SupplyContractChart.vue](src/components/dashboard/SupplyContractChart.vue)

#### 1.1 删除顶部自定义图例（第21-26行）
```html
<!-- 删除 -->
<div class="flex flex-wrap gap-4 mb-3">
  <div class="flex items-center text-xs" v-for="item in legendItems" :key="item.name">
    <span class="w-3 h-3 rounded mr-1" :style="{ backgroundColor: item.color }"></span>
    <span class="text-gray-600">{{ item.name }}</span>
  </div>
</div>
```

#### 1.2 柱状图模式启用 ECharts 内置 legend（第200-202行）
```js
// 改前
legend: { show: false }
// 改后
legend: {
  show: true,
  bottom: 0,
  itemWidth: 14,
  itemHeight: 10,
  textStyle: { fontSize: 11 }
}
```

#### 1.3 grid.bottom 调整（第203-208行）
`bottom: '3%'` → `bottom: '15%'` 给底部图例留空间

#### 1.4 图表容器改为 flex-1 铺满
- 第28行 `<div ref="chartRef" class="h-64">` → `<div ref="chartRef" class="flex-1 min-h-[200px]">`
- 外层 div 已有 `h-full`，配合 flex-1 可自动撑满

### 调整2：各子组件铺满空白

**统一策略**：将所有图表容器的固定高度 `h-64` 改为弹性布局 `flex-1 min-h-[200px]`

| 组件 | 文件 | 当前 | 改为 |
|------|------|------|------|
| SupplierStatChart | [SupplierStatChart.vue#L40](src/components/dashboard/SupplierStatChart.vue#L40) | `class="h-64"` | `class="flex-1 min-h-[200px]"` |
| MaterialProcurementChart | [MaterialProcurementChart.vue#L56](src/components/dashboard/MaterialProcurementChart.vue#L56) | `class="h-64"` | `class="flex-1 min-h-[200px]"` |
| SupplyContractChart | [SupplyContractChart.vue#L28](src/components/dashboard/SupplyContractChart.vue#L28) | `class="h-64"` | `class="flex-1 min-h-[200px]"` |

### 调整3：Dashboard 父级容器支持拉伸

**文件**: [SupplyChainDashboard.vue](src/views/SupplyChainDashboard.vue)

各 grid 子项的包裹 div 已有正确的类名：
- 第31行 `<div class="lg:col-span-2">` — 内含 SupplyChainTimelines（已有 flex 布局）
- 第46行 `<div class="lg:col-span-1">` — 内含 SupplyContractChart（需确认 h-full 生效）
- 第60行 `<div class="lg:col-span-2">` — 内部是白色卡片 + 两个 SupplierStatChart
- 第89行 `<div class="lg:col-span-1 flex flex-col">` — 已有 flex flex-col

需要确保每个包含图表的容器都有 `flex flex-col` 和 `h-full` 或 `min-h`。

具体修改：

1. **第31行**：`<div class="lg:col-span-2">` → `<div class="lg:col-span-2 flex flex-col">`
2. **第46行**：`<div class="lg:col-span-1">` → `<div class="lg:col-span-1 flex flex-col">`
3. **第62行**：供应商统计的白色卡片加 `flex flex-col h-full`
4. **第69行**：供应商统计的 grid 加 `flex-1`
