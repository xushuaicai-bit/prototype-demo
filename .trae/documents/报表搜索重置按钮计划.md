# 所有报表增加搜索和重置按钮

## 概述
为所有带筛选区域的报表组件，在筛选栏末尾添加"搜索"和"重置"两个按钮。

---

## 当前状态分析

### 有筛选区的报表（需加按钮）

| 报表 | 文件 | 筛选字段 | 当前按钮 |
|------|------|---------|---------|
| 营收汇总表 | [RevenueSummary.vue](src/components/revenue/RevenueSummary.vue) | 基层单位(多选) | "清空筛选"(仅清空单位) |
| 营收明细表 | [RevenueDetail.vue](src/components/revenue/RevenueDetail.vue) | 单位/状态/口径/协管/项目名/编号/下调率/预警等级 | **无** |
| 偏差项目表 | [RevenueDeviation.vue](src/components/revenue/RevenueDeviation.vue) | 项目编号/名称/单位/偏差类型 | **无** |
| 供应商合同签订 | [SupplierContractReport.vue](src/components/supply-chain/SupplierContractReport.vue) | 项目名称/基层单位 | **无** |
| 项目采购合同 | [SupplierDetailReport.vue](src/components/supply-chain/SupplierDetailReport.vue) | 供应商名称/合同分类 | **无** |

### 无筛选区的报表（市场报表，纯表格）
- BidSummaryReport / InBidReport / ProjectTrackingReport / KeyProjectListReport — 不需要添加

---

## 修改方案

每个报表的筛选区域（`bg-gray-50 rounded-lg` 的 div）末尾，统一追加两个按钮：

```html
<!-- 在筛选区域最后一个字段后面 -->
<div class="flex items-center gap-2 ml-auto">
  <button 
    class="flex items-center px-4 py-1.5 bg-blue-500 text-white text-sm rounded hover:bg-blue-600 transition-colors"
    @click="handleSearch"
  >
    <svg class="w-4 h-4 mr-1" fill="none" stroke="currentColor" viewBox="0 0 24 24">
      <path stroke-linecap="round" stroke-linejoin="round" stroke-width="2" d="M21 21l-6-6m2-5a7 7 0 11-14 0 7 7 0 0114 0z" />
    </svg>
    搜索
  </button>
  <button 
    class="flex items-center px-4 py-1.5 bg-white border border-gray-300 text-gray-700 text-sm rounded hover:bg-gray-50 transition-colors"
    @click="handleReset"
  >
    <svg class="w-4 h-4 mr-1" fill="none" stroke="currentColor" viewBox="0 0 24 24">
      <path stroke-linecap="round" stroke-linejoin="round" stroke-width="2" d="M4 4v5h.582m15.356 2A8.001 8.001 0 004.582 9m0 0H9m11 11v-5h-.581m0 0a8.003 8.003 0 01-15.357-2m15.357 2H15" />
    </svg>
    重置
  </button>
</div>
```

### 各报表具体改动

#### 1. RevenueSummary.vue（L22-57 筛选区）
- 将原有 L49-55 的"清空筛选"按钮替换为搜索+重置
- `handleSearch`: 触发过滤（已有 filteredData computed，只需确认生效）
- `handleReset`: `selectedUnits = []`（清空多选）

#### 2. RevenueDetail.vue（L22-93 筛选区）
- 在 `</div>` 关闭筛选区前（L93后）插入按钮组
- `handleReset`: 重置所有 filters 字段为默认值（`filters = { unit:'', status:'', ... }`）

#### 3. RevenueDeviation.vue（L29-? 筛选区）
- 同上，在筛选区末尾插入按钮组
- `handleReset`: 重置 filters 为默认值

#### 4. SupplierContractReport.vue（L25-? 筛选区）
- 同上，`handleReset`: `selectedProject = ''`, `selectedUnit = ''`

#### 5. SupplierDetailReport.vue（L26-? 筛选区）
- 同上，`handleReset`: `selectedSupplier = ''`, `selectedCategory = ''`

### 按钮样式规范
- **搜索**: `bg-blue-500 text-white` + 放大镜图标
- **重置**: `bg-white border border-gray-300 text-gray-700` + 刷新图标
- 使用 `ml-auto` 靠右对齐
- 尺寸: `px-4 py-1.5 text-sm rounded`（与现有筛选控件协调）
