# 修复柱子高度 + 纵轴标签重叠 + 材料集采高度对齐

## 问题分析

1. **柱子太矮**：`min-h-[200px]` 在 flex-1 模式下被压缩，图表实际可用高度不足
2. **纵轴数字叠在一起**：ECharts yAxis 默认自动计算刻度间隔，小容器内标签重叠；需设置 `axisLabel.interval: 0` 强制全部显示或调整字体/间距
3. **材料集采与Top10高度不一致**：两者 min-h 相同但内部内容不同（材料集采有 Tab + 类别按钮 + 摘要栏），导致图表区域高度不同

---

## 修改方案

### 调整1：SupplierStatChart — 增加最小高度 + 纵轴防重叠

**文件**: [SupplierStatChart.vue](src/components/dashboard/SupplierStatChart.vue)

#### 1.1 图表容器增加最小高度
- 第34行：`class="flex-1 min-h-[200px]"` → `class="flex-1 min-h-[300px]"`

#### 1.2 yAxis 配置加 axisLabel 防重叠
在两个 yAxis 配置中添加：
```js
axisLabel: {
  fontSize: 11,
  interval: 0        // 强制显示所有标签（不跳过）
}
```
具体位置：
- A级承包商模式：第114行左侧Y轴、第159-166行右侧Y轴
- Top10模式：第169行左侧Y轴、第170-175行右侧Y轴

### 调整2：MaterialProcurementChart — 高度对齐 + 纵轴防重叠

**文件**: [MaterialProcurementChart.vue](src/components/dashboard/MaterialProcurementChart.vue)

#### 2.1 图表容器增加最小高度
- 第56行：`class="flex-1 min-h-[200px]"` → `class="flex-1 min-h-[300px]"`

#### 2.2 yAxis 加 axisLabel 防重叠
- 第164-168行左侧Y轴、第169-174行右侧Y轴，加 `fontSize: 11, interval: 0`

### 调整3：SupplyContractChart — 同步增加高度（可选）

如果合同签订也需要一致高度：
**文件**: [SupplyContractChart.vue](src/components/dashboard/SupplyContractChart.vue)
- 第21行：`class="flex-1 min-h-[200px]"` → `class="flex-1 min-h-[300px]"`
