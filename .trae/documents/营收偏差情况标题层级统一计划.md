## 改造方案

### UnifiedDashboard.vue L280-283
将 h3 改为与 L224 一致的层级样式，去掉小蓝条 span：

**原**：
```html
<h3 class="text-sm font-semibold text-gray-800 flex items-center">
  <span class="w-1 h-4 bg-blue-500 rounded-full mr-2"></span>
  {{ selectedYear }}年{{ selectedMonth }}月营收偏差情况
</h3>
```

**改后**：
```html
<h3 class="text-base font-semibold text-gray-800">
  {{ selectedYear }}年{{ selectedMonth }}月营收偏差情况
</h3>
```

变更点：
1. 删除 `<span class="w-1 h-4 bg-blue-500 rounded-full mr-2"></span>`（小蓝条）
2. `text-sm` → `text-base`（与"营收计划上报及时性"同字号）
3. 去掉 `flex items-center`（不再需要，因无内联 span）

## 不变更项

- "营收计划上报及时性"标题本身
