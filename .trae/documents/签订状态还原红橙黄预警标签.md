# 签订状态列还原红橙黄预警标签计划

## Summary

将"签订状态"列从简单的"已签订/未签订"文字还原为：**已签订显示绿色，未签订时根据超期天数显示红/橙/黄预警标签**。

## Current State Analysis

### 当前签订状态列（L198-L209）

```html
<el-table-column label="签订状态" prop="signStatus" width="100">
  <template #default="scope">
    <span :class="scope.row.signStatus === '已签订' ? 'text-green-600 font-medium' : 'text-orange-500 font-medium'">
      {{ scope.row.signStatus }}
    </span>
  </template>
</el-table-column>
```

问题：未签订统一显示橙色"未签订"，没有区分超期程度。

### 已有可复用的函数（L1157-L1173）

```javascript
// 预警样式：红(>=60天)/橙(45-60天)/黄(21-44天)
const getSignStatusClass = (row) => {
  if (row.signStatus === '已签订') return ''
  const days = row.overdueDays
  if (days >= 60) return 'bg-red-100 text-red-800 px-2 py-1 rounded'
  if (days >= 45) return 'bg-orange-100 text-orange-800 px-2 py-1 rounded'
  if (days >= 21) return 'bg-yellow-100 text-yellow-800 px-2 py-1 rounded'
  return ''
}

// 预警标签文字
const getWarningLabel = (row) => {
  const days = row.overdueDays
  if (days >= 60) return '红色预警'
  if (days >= 45) return '橙色预警'
  if (days >= 21) return '黄色预警'
  return '未签订'
}
```

这两个函数已经存在且逻辑正确，只是当前模板没调用它们。

## Proposed Changes

### 文件：`e:\trae_demo_env\营收看板\src\components\supply-chain\SupplierContractReport.vue`

仅修改 **L204-L208** 的 `<template #default>` 内容：

```html
<template #default="scope">
  <span v-if="scope.row.signStatus === '已签订'" class="text-green-600 font-medium">已签订</span>
  <span v-else :class="getSignStatusClass(scope.row)">{{ getWarningLabel(scope.row) }}</span>
</template>
```

**效果**：
| signStatus | overdueDays | 显示 | 样式 |
|-----------|-------------|------|------|
| 已签订 | - | 已签订 | 绿色文字 |
| 未签订 | 0~20 | 未签订 | 无背景（普通文字） |
| 未签订 | 21~44 | 黄色预警 | 黄底黄字圆角标签 |
| 未签订 | 45~60 | 橙色预警 | 橙底橙字圆角标签 |
| 未签订 | 61+ | 红色预警 | 红底红字圆角标签 |
