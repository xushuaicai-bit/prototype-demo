# 恢复材料集采图表点击跳转

## 摘要
在 MaterialProcurementChart.vue 的 initChart 中恢复 chart.on('click') 跳转外部链接。

## 改动
文件：`e:\trae_demo_env\营收看板\src\components\dashboard\MaterialProcurementChart.vue`

在 `chart.setOption(option)` 之后添加：
```js
chart.on('click', () => {
  window.open('https://www.smart-worksite.com/hj/#/micro/supplies/biz/enterprise/supplies/frame-contract/index', '_blank')
})
```

## 验证
点击图表柱子/折线，确认跳转至外部链接
