## 三、具体改动清单

### 改动1: 两模块比例 — 3/4 : 1/4（即9:3）

**L33**: `col-span-10` → **`col-span-9`**
**L136**: `col-span-2` → **`col-span-3`**

### 改动2: '其他业态类' → '其他类'

**L563**: yearTotalRevenue 数组第3项
```javascript
// 当前
{ name: '其他业态类', value: '1,000万元' }
// 改为
{ name: '其他类', value: '1,000万元' }
```

### 改动3: 去掉所有"小模块"

共4处标题文字修改：

| 行号 | 当前 | 改为 |

### 改动4: 及时性+偏差合并保持不变

无需操作。

### 改动5: 三图表区域改为 cols-3

**L196**: `grid grid-cols-1 lg:grid-cols-2 gap-4` → **`grid grid-cols-1 lg:grid-cols-3 gap-4`**

当前结构：
```
[cols-2]
  [合并模块：及时性+偏差]   [月度统计]   [累计营收]  ← 累计营收换行了
```
目标：
```
[cols-3]
  [合并模块：及时性+偏差]   [月度统计]   [累计营收]  ← 三列等宽并排
```
