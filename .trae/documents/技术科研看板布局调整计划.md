# 技术科研看板布局调整计划

## 概述
调整技术科研看板的卡片布局：第一行放3个卡片（施工方案、知识库使用情况、基层单位科研项目数量），第二行放2个卡片（科研成果统计、科研费用管控）。

## 涉及文件
- `e:\trae_demo_env\营收看板\src\views\TechResearchDashboard.vue` — 仅修改 template 部分的布局结构

## 当前状态分析

当前布局（3行）：
1. **第一行**（第28-101行，`grid-cols-1 lg:grid-cols-2`）：施工方案 + 知识库使用情况
2. **第二行**（第103-154行，`grid-cols-1 lg:grid-cols-2`）：基层单位科研项目数量 + 科研成果统计
3. **第三行**（第156-187行，单独全宽）：科研费用管控

## 修改方案

将布局重组为2行：

### 修改1：第一行改为3列，包含施工方案、知识库使用情况、基层单位科研项目数量

将第28行的 `lg:grid-cols-2` 改为 `lg:grid-cols-3`，并将"基层单位科研项目数量"卡片（当前第104-115行）移入此 grid 容器，放在"知识库使用情况"之后。

### 修改2：第二行保持2列，包含科研成果统计、科研费用管控

将"科研费用管控"卡片（当前第156-187行）移入第二行 grid 容器（`lg:grid-cols-2`），放在"科研成果统计"之后。

### 修改3：删除原来单独的第三行容器

"科研费用管控"已移入第二行，删除其外层独立的 `<div class="bg-white rounded-xl p-4 shadow-sm">` 包裹（实际上保留卡片本身，只是移位置）。

### 最终结构

```html
<!-- 第一行：3列 -->
<div class="grid grid-cols-1 lg:grid-cols-3 gap-4">
  <!-- 施工方案卡片 -->
  <!-- 知识库使用情况卡片 -->
  <!-- 基层单位科研项目数量卡片 -->
</div>

<!-- 第二行：2列 -->
<div class="grid grid-cols-1 lg:grid-cols-2 gap-4">
  <!-- 科研成果统计卡片 -->
  <!-- 科研费用管控卡片 -->
</div>
```

## 假设与决策
1. **不调整卡片内部内容**：仅移动卡片位置和改变 grid 列数，不修改卡片内部的图表配置、高度等。
2. **响应式断点**：大屏（lg）下第一行3列、第二行2列；小屏下均为1列堆叠，保持原有 `grid-cols-1` 前缀。
3. **卡片样式不变**：所有卡片保持原有的 `bg-white rounded-xl p-4 shadow-sm` 样式。
