### 调整4：标题重命名

**文件**: [mockData.js](src/data/mockData.js)
- **第248行**: `title: '材料集采情况'` → `title: '材料集采合同情况'`

**文件**: [MaterialProcurementChart.vue](src/components/dashboard/MaterialProcurementChart.vue)

当前标题动态替换逻辑（第73-75行）：
```js
const currentTitle = computed(() => {
  return props.title?.replace('集采', activeTab.value) || props.title
})
```

当 activeTab='集租' 时，`'材料集采合同情况'.replace('集采', '集租')` = `'材料集租合同情况'`，符合需求。

但用户要求的是 `'设备集租合同情况'`，不是简单的字符串替换。需要调整逻辑：

```js
const currentTitle = computed(() => {
  if (activeTab.value === '集采') {
    return props.title || ''  // 显示"材料集采合同情况"
  }
  return '设备集租合同情况'  // 集租时显示"设备集租合同情况"
})
```

## 假设与决策

1. **未签柱状图颜色**：使用灰色 `#9ca3af` 表示"未签"，区别于绿色（已完成）和红橙黄（预警）
2. **堆叠顺序**：未签(底) → 黄色预警 → 橙色预警 → 红色预警(顶)，符合视觉层次
3. **集采类别字段**：以按钮组形式展示在 Tab 旁边，不改变现有图表数据（Mock数据暂不按类别细分，仅展示UI框架）
4. **标题替换逻辑**：集租标题硬编码为"设备集租合同情况"，而非简单字符串替换
