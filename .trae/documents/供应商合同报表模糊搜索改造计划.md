# 供应商合同签订报表 — 项目名称/供应商名称改为模糊搜索

## 摘要
将 `SupplierContractReport.vue` 筛选栏中的"项目名称"和"供应商名称"从 el-select 下拉精确匹配改为 el-input 模糊搜索，参照项目中 `RevenueDetail.vue` 已有的模糊搜索模式。

## 当前状态分析
- 文件：`e:\trae_demo_env\营收看板\src\components\supply-chain\SupplierContractReport.vue`
- 项目名称：L28-40 使用 `<el-select>` + `allProjects` 静态数组（L413-423），过滤逻辑 L1156 使用 `!==` 精确匹配
- 供应商名称：L96-108 使用 `<el-select>` + `allSuppliers` computed（L1163-1171），过滤逻辑 L1186 使用 `!==` 精确匹配
- `selectedProject` ref（L402）、`selectedSupplier` ref（L407）已存在，继续复用
- `handleReset`（L1280-1290）已重置为空字符串，无需修改

## 改造方案
参照 `RevenueDetail.vue`（L69-77 模板、L1895-1900 过滤）的 el-input + `.includes()` 模式。

### 改动清单

#### 1. 模板：项目名称 el-select → el-input（L26-41）
```html
<!-- 改前 -->
<el-select v-model="selectedProject" placeholder="请选择项目名称" class="w-64" clearable>
  <el-option v-for="project in allProjects" :key="project" :label="project" :value="project" />
</el-select>

<!-- 改后 -->
<el-input v-model="selectedProject" placeholder="请输入项目名称" class="w-48" clearable />
```

#### 2. 模板：供应商名称 el-select → el-input（L94-109）
```html
<!-- 改前 -->
<el-select v-model="selectedSupplier" placeholder="请选择供应商" class="w-64" clearable>
  <el-option v-for="s in allSuppliers" :key="s" :label="s" :value="s" />
</el-select>

<!-- 改后 -->
<el-input v-model="selectedSupplier" placeholder="请输入供应商名称" class="w-48" clearable />
```

#### 3. 过滤逻辑：精确匹配 → 模糊匹配
- **L1156**（filteredData computed）：`project.projectName !== selectedProject.value` → `!project.projectName.includes(selectedProject.value)`
- **L1186**（tableData computed）：`contract.supplierName !== selectedSupplier.value` → `!contract.supplierName.includes(selectedSupplier.value)`

#### 4. 清理无用代码
- 删除 `allProjects` 静态数组（L413-423）— 不再需要选项列表
- 删除 `allSuppliers` computed（L1163-1171）— 不再需要选项列表

## 假设与决策
