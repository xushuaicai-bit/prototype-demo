## 问题1：营收汇总表字段缺失

### 现状分析
**文件**: [RevenueSummary.vue](src/components/revenue/RevenueSummary.vue)

模板中 L260-290 的列定义**存在且正确**：
- `label="上报营收及剩余合同存量"` 分组下有3个子列：
  - "本年上报股份营收" (prop=`reportedRevenue`)
  - "新接项目合同额（去税）" (prop=`newContractAmount`)
  - "截止本月剩余合同存量" (prop=`remainingContract`)

底部合计行（L325-L327）也有对应字段。

**可能原因**：rawData 数据对象中缺少这些字段的值 → 表格显示为空/ `-`

### 修改方案
检查 rawData 中 constructionData 的每个对象是否包含 `reportedRevenue`、`newContractAmount`、`remainingContract` 字段。如果缺少则补充数据。

## 问题2：当年累计完成营收纵轴标识反了

### 现状分析
**文件**: [UnifiedDashboard.vue](src/views/UnifiedDashboard.vue) 的 `initSectorChart()` 函数

**当前配置 (L991-L994)**:
```js
yAxis: [
  { type: 'value', name: '累计营收(万元)', position: 'left' },    // ← 左边
  { type: 'value', name: '项目数量', position: 'right' }           // ← 右边
]
```

**series 绑定 (L996-L1014)**:
```js
{ name: '累计营收', type: 'line', yAxisIndex: 1, ... }  // 折线→右轴(项目数量?)
{ name: '项目数量', type: 'bar', yAxisIndex: 0, ... }     // 柱子→左轴(累计营收?)
```

**矛盾**：名称和绑定不匹配！柱子(项目数量)绑到左轴但左轴叫"累计营收"，折线(累计营收)绑到右轴但右轴叫"项目数量"。两边都反了。

### 修改方案
交换 yAxis 的 name 和 position，同时交换 series 的 yAxisIndex：

```js
// 改后:
yAxis: [
  { type: 'value', name: '项目数量', position: 'left' },
  { type: 'value', name: '累计营收(万元)', position: 'right' }
]
series: [
  { name: '累计营收', type: 'line', yAxisIndex: 1, ... },  // 折线→右轴=累计营收 ✓
  { name: '项目数量', type: 'bar', yAxisIndex: 0, ... }     // 柱子→左轴=项目数量 ✓
]
```

## 问题3：营收明细表年月筛选+当月/上月跟随

### 现状分析
**文件**: [RevenueDetail.vue](src/components/revenue/RevenueDetail.vue)

**当前 (L447-L448)**:
```js
detailYear: '2026',
detailMonth: ''          // ← 空，未选中任何月
```

**动态标签 (L432-L434)**:
```js
const dynamicMonthLabel = computed(() => {
  const m = filters.value.detailMonth
  return m ? `${m}月` : '当月'    // detailMonth为空时显示"当月"
})
```

用户要求：默认选2026年6月，当月/上月字段也跟随显示对应月份的数据。

### 修改方案
1. `detailMonth` 默认值从 `''` 改为 `6`
2. 检查表格中涉及"当月"、"上月"的列（如 `monthActualRevenue` 等），确认它们是否根据 `detailMonth` 动态计算或只是静态显示
3. 如果是静态的"当月实际营收"等列名，可能需要改为动态标题如"X月实际营收"

## 问题4：营收月度统计时间显示不全

### 现状分析
**文件**: [UnifiedDashboard.vue](src/views/UnifiedDashboard.vue) 的 `initCharts()` 函数中的营收图表

**当前 xAxis data (L938)**:
```js
data: ['2025-11', '2025-12', '2026-01', '2026-02', '2026-03', '2026-04']
```
只有6个时间点，且最新只到2026-04。当前日期是2026年6月，数据应该延伸到至少2026-05或2026-06。

**xAxis 配置**:
```js
axisLabel: {
  interval: 0,
  rotate: 35,
  fontSize: 10,
  margin: 10
}
```
rotate: 35 已经倾斜了，但如果容器宽度不够仍可能被截断。

### 修改方案
1. 扩展 xAxis data 增加更多月份（如到 2026-06）
2. 同步扩展每个 series 的 data 数组长度
3. 可调整 grid.left/right 或 axisLabel.fontSize 给时间标签更多空间
