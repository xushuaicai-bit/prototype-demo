### 修改3：图表增加右侧纵轴和折线（updateTechChart 函数，第305-352行）

为三个图表分别配置：
- approval 图表：右侧纵轴名称"审批完成率"，折线显示审批完成率
- scheme 图表：右侧纵轴名称"退回率"，折线显示退回率
- disclosure 图表：右侧纵轴名称"交底率"，折线显示交底率

**折线数据计算方式**（从现有 series 数据动态计算百分比）：
- approval 审批完成率 = 完成数 / (完成数 + 未完成数) × 100
- scheme 退回率 = 退回数 / (退回数 + 通过数) × 100
- disclosure 交底率 = 已交底 / (已交底 + 未交底) × 100

**修改后的 updateTechChart 函数：**

```javascript
const updateTechChart = () => {
  if (!techChart) return

  const data = currentTechData.value
  const colors = {
    approval: ['#3b82f6', '#60a5fa'],
    scheme: ['#f59e0b', '#fbbf24'],
    disclosure: ['#10b981', '#6ee7b7']
  }

  // 右侧纵轴名称及折线名称配置
  const rateConfig = {
    approval: { name: '审批完成率', color: '#ef4444' },
    scheme: { name: '退回率', color: '#8b5cf6' },
    disclosure: { name: '交底率', color: '#f59e0b' }
  }

  // 根据当前 tab 计算比率数据（折线数据）
  const rateData = data.series[0].data.map((val, i) => {
    const total = val + data.series[1].data[i]
    return total > 0 ? Number(((val / total) * 100).toFixed(1)) : 0
  })

  const currentRate = rateConfig[activeTechTab.value]

  const option = {
    tooltip: {
      trigger: 'axis',
      axisPointer: {
        type: 'shadow'
      }
    },
    legend: {
      data: [...data.series.map(s => s.name), currentRate.name]
    },
    grid: {
      left: '3%',
      right: '6%',
      bottom: '3%',
      top: '15%',
      containLabel: true
    },
    xAxis: {
      type: 'category',
      data: data.categories,
      axisLabel: {
        interval: 0,
        rotate: 30,
        fontSize: 10
      }
    },
    yAxis: [
      {
        type: 'value',
        name: '数量'
      },
      {
        type: 'value',
        name: currentRate.name,
        axisLabel: {
          formatter: '{value}%'
        },
        max: 100,
        min: 0
      }
    ],
    series: [
      ...data.series.map((s, i) => ({
        name: s.name,
        type: 'bar',
        data: s.data,
        itemStyle: {
          color: colors[activeTechTab.value][i]
        }
      })),
      {
        name: currentRate.name,
        type: 'line',
        yAxisIndex: 1,
        data: rateData,
        itemStyle: {
          color: currentRate.color
        },
        line: {
          width: 2
        },
        symbol: 'circle',
        symbolSize: 6
      }
    ]
  }

  techChart.setOption(option, true)
}
```

**关键变更点说明：**
1. `yAxis` 从单个对象改为数组，包含左侧数值轴（数量）和右侧百分比轴
2. 右侧纵轴 `axisLabel.formatter` 设为 `{value}%`，max 设为 100
3. `series` 在柱状图基础上追加一个 `type: 'line'` 折线，`yAxisIndex: 1` 绑定右侧纵轴
4. 增加 `legend` 显示折线名称
5. `grid.right` 调整为 `'6%'` 给右侧纵轴留出空间，`grid.top` 调整为 `'15%'` 给 legend 留出空间
6. `setOption(option, true)` 使用 `notMerge=true`，确保切换 tab 时旧 series 被清除
7. 折线颜色：审批完成率红色(#ef4444)、退回率紫色(#8b5cf6)、交底率橙色(#f59e0b)，与柱状图颜色区分

## 假设与决策
1. **折线数据来源**：从现有柱状图数据动态计算百分比，不修改 mockData.js。因为现有数据已包含计算所需的分子和分母。
2. **折线颜色**：选择与柱状图对比鲜明的颜色（红/紫/橙），便于区分。
3. **右侧纵轴范围**：0-100%，因为比率的合理范围是 0-100%。
4. **legend 显示**：增加 legend 以便用户区分柱状图和折线的含义。
5. **setOption notMerge**：使用 `true` 确保切换 tab 时图表完全刷新，避免 series 残留。
