# 安全子看板进入专版图片跳转计划

## 一、需求概述

在安全在线图片（safe-board.png）上，点击三个"进入专版"区域时分别跳转到不同图片：
- 实名制 → safe-1.png
- 隐患排查 → safe-2.png
- 视频监控 → safe-3.png

需要提供返回按钮回到主图。

## 二、当前状态分析

- [SafetyDashboard.vue](file:///e:/trae_demo_env/营收看板/src/views/SafetyDashboard.vue) 有两个Tab（安全在线/过程管控），通过 `activeTab` 切换图片
- 图片使用 `object-contain` 自适应显示
- public 目录已有 safe-board.png 和 safe-daily.png

## 三、实现步骤

### 步骤1：复制三张图片到 public 目录
- `D:\...\safe-1.png` → `public\safe-1.png`
- `D:\...\safe-2.png` → `public\safe-2.png`
- `D:\...\safe-3.png` → `public\safe-3.png`

### 步骤2：修改 SafetyDashboard.vue
新增 `view` 状态控制当前视图，在"安全在线"Tab下：
- `view = 'main'`（默认）：显示 safe-board.png，图片上方叠加三个"进入专版"按钮（实名制、隐患排查、视频监控）
- `view = 'realname'`：显示 safe-1.png，左上角显示"返回"按钮
- `view = 'hazard'`：显示 safe-2.png，左上角显示"返回"按钮
- `view = 'video'`：显示 safe-3.png，左上角显示"返回"按钮

三个"进入专版"按钮使用绝对定位叠加在图片底部（三个分区位置），使用半透明样式。切换Tab时重置 view 为 'main'。

## 四、验证
1. 安全在线Tab默认显示主图 + 三个进入专版按钮
2. 点击实名制→显示 safe-1.png，点返回→回主图
3. 点击隐患排查→显示 safe-2.png，点返回→回主图
4. 点击视频监控→显示 safe-3.png，点返回→回主图
5. 过程管控Tab不受影响
