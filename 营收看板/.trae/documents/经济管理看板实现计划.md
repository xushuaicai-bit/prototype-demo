# 经济管理看板实现计划

## 一、需求分析

### 目标

在经济管理模块下添加"经济管理看板"菜单项，点击后通过 iframe 嵌入外部链接：

* **URL**: `https://www.smart-worksite.com/hj/#/micro/economic/biz/ent/economy/dashboard`

* **要求**: 只显示主界面内容，不显示该系统的功能菜单

### 当前状态

* 经济管理模块目前为空状态，显示"该模块正在开发中"

* 需要创建新的看板视图组件

## 二、技术方案

### 方案选择：iframe 嵌入

使用 `<iframe>` 标签嵌入外部页面，这是最直接的方式。

**优点**:

* 实现简单快速

* 完整保留原系统界面和交互

* 无需了解外部系统内部结构

**注意事项**:

* 外部系统需要支持跨域嵌入

* 可能需要处理登录认证问题

* 需要隐藏外部系统的菜单栏

## 三、实施步骤

### 步骤 1: 创建 EconomyDashboard.vue 组件

**文件**: `src/views/EconomyDashboard.vue`

**功能实现**:

```vue
<template>
  <div class="h-full flex flex-col">
    <!-- 页面标题 -->
    <div class="bg-white rounded-xl p-4 shadow-sm mb-4">
      <div class="flex items-center gap-3">
        <svg class="w-6 h-6 text-green-600" fill="none" stroke="currentColor" viewBox="0 0 24 24">
          <path stroke-linecap="round" stroke-linejoin="round" stroke-width="2" d="M9 19v-6a2 2 0 00-2-2H5a2 2 0 00-2 2v6a2 2 0 002 2h2a2 2 0 002-2zm0 0V9a2 2 0 012-2h2a2 2 0 012 2v10m-6 0a2 2 0 002 2h2a2 2 0 002-2m0 0V5a2 2 0 012-2h2a2 2 0 012 2v14a2 2 0 01-2 2h-2a2 2 0 01-2-2z"></path>
        </svg>
        <h2 class="text-lg font-semibold text-gray-800">经济管理看板</h2>
      </div>
    </div>
    
    <!-- iframe 嵌入区域 -->
    <div class="flex-1 bg-white rounded-xl shadow-sm overflow-hidden">
      <iframe 
        ref="iframeRef"
        :src="iframeUrl"
        class="w-full h-full border-0"
        frameborder="0"
        allowfullscreen
        @load="onIframeLoad"
      ></iframe>
      
      <!-- 加载状态 -->
      <div v-if="loading" class="absolute inset-0 flex items-center justify-center bg-white bg-opacity-80">
        <div class="text-center">
          <div class="animate-spin rounded-full h-12 w-12 border-b-2 border-blue-600 mx-auto mb-3"></div>
          <p class="text-gray-600">正在加载经济管理数据...</p>
        </div>
      </div>
      
      <!-- 错误状态 -->
      <div v-if="error" class="absolute inset-0 flex items-center justify-center bg-gray-50">
        <div class="text-center p-8">
          <svg class="w-16 h-16 text-red-400 mx-auto mb-4" fill="none" stroke="currentColor" viewBox="0 0 24 24">
            <path stroke-linecap="round" stroke-linejoin="round" stroke-width="2" d="M12 8v4m0 4h.01M21 12a9 9 0 11-18 0 9 9 0 0118 0z"></path>
          </svg>
          <h3 class="text-lg font-medium text-gray-800 mb-2">加载失败</h3>
          <p class="text-gray-600 mb-4">{{ error }}</p>
          <button 
            @click="reloadIframe"
            class="px-6 py-2 bg-blue-600 text-white rounded-lg hover:bg-blue-700 transition-colors"
          >
            重新加载
          </button>
        </div>
      </div>
    </div>
  </div>
</template>

<script setup>
import { ref, onMounted } from 'vue'

const iframeRef = ref(null)
const loading = ref(true)
const error = ref('')

const iframeUrl = 'https://www.smart-worksite.com/hj/#/micro/economic/biz/ent/economy/dashboard'

const onIframeLoad = () => {
  loading.value = false
  error.value = ''
}

const reloadIframe = () => {
  loading.value = true
  error.value = ''
  if (iframeRef.value) {
    iframeRef.value.src = iframeUrl
  }
}

// 错误处理
onMounted(() => {
  setTimeout(() => {
    if (loading.value) {
      error.value = '加载超时，请检查网络连接或稍后重试'
      loading.value = false
    }
  }, 30000) // 30秒超时
})
</script>
```

**关键特性**:

* ✅ 全屏 iframe 嵌入

* ✅ 加载状态提示

* ✅ 错误处理和重试机制

* ✅ 超时检测（30秒）

* ✅ 响应式布局

## 六、潜在风险与应对

### 风险1: 跨域限制

**现象**: iframe 无法加载或显示空白
**应对**:

* 检查目标网站是否设置 `X-Frame-Options` 头

* 如有限制，考虑联系对方开放白名单

* 备选方案：使用反向代理

### 风险2: 登录认证

**现象**: 显示登录页面而非看板内容
**应对**:

* 如果需要登录：提示用户先在外部系统登录

* 或传递 token 参数（需对方支持）

### 风险3: 性能影响

**现象**: 页面加载缓慢
**应对**:

* 已添加加载状态提示

* 可考虑懒加载策略

* 设置合理的超时时间
