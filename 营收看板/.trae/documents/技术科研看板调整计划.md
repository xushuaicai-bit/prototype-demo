# 技术科研看板调整计划

## 1. 需求分析

根据用户需求，需要对技术科研看板进行以下调整：

1. **技术子模块整合**：
   - 审批完成率、方案退回率、交底情况三个图表用tab切换
   - 横轴支持在9家基层单位和8大业态之间切换
   - 每个图表下方有相应的总结文字

2. **图表组件替换**：
   - 所有柱状图改用element组件和echarts实现

3. **布局调整**：
   - 技术子模块（审批完成率+方案退回率+交底情况+知识库使用情况）放在一行
   - 科研子模块（基层单位科研项目数量+科研成果统计+科研费用管控）放在一排
   - 科研成果统计和科研成果分布合并为一个图表，用标签切换
   - 科研费用管控仅显示当前季度情况，其他季度用右上角下拉筛选

4. **UI优化**：保持整体美观和自适应

## 2. 实现方案

### 2.1 文件修改

| 文件路径 | 修改内容 |
|---------|---------|
| `src/views/TechResearchDashboard.vue` | 完全重写，实现新的布局和功能 |
| `src/data/mockData.js` | 添加新的数据结构，支持8大业态、科研费用季度数据等 |
| `package.json` | 确保已安装echarts和element-plus（如果需要） |

### 2.2 组件设计

1. **技术子模块 (TechModule)**：
   - 顶部三个tab：审批完成率、方案退回率、交底情况
   - 图表顶部横轴切换：9家基层单位 / 8大业态
   - 使用echarts实现柱状图
   - 下方显示总结文字

2. **知识库组件 (KnowledgeBase)**：
   - 保持现有数据展示形式
   - 可以考虑优化为图表展示

3. **科研成果统计 (ResearchAchievements)**：
   - 两个tab：按成果类型 / 按基层单位
   - 使用echarts实现柱状图和饼图

4. **科研项目组件 (ResearchProjects)**：
   - 9家基层单位科研项目数量柱状图

5. **科研费用组件 (ResearchExpenses)**：
   - 右上角季度下拉选择
   - 展示当前季度费用数据

### 2.3 数据结构

需要在 `mockData.js` 中添加：
- 8大业态数据结构
- 各业态审批完成率、方案退回率、交底情况数据
- 科研费用各季度详细数据

## 3. 实施步骤

1. 更新 `mockData.js` 添加所需数据
2. 重写 `TechResearchDashboard.vue` 实现新布局
3. 整合echarts组件实现图表功能
4. 添加tab切换逻辑
5. 优化响应式布局
6. 测试和调整
