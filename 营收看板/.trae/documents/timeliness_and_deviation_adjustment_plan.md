# 营收计划上报及时性和偏差情况调整计划

## 需求概述

### 1. 营收计划上报及时性调整
- **图例颜色修改**：
  - 🔵 蓝色 = 应上报
  - 🟢 绿色 = 已上报
  - 🟡 **浅黄色** = 待上报（当前为橙色）
  
- **删除按钮**：移除图表下方的红色"应上报"和绿色"已上报"按钮

### 2. 4月营收偏差情况布局调整
- **当前问题**：竖向下拉显示，比例不协调
- **目标**：每个公司分模块横向显示

## 详细实施步骤

### 第一步：修改营收计划上报及时性

#### 1.1 修改"待上报"颜色为浅黄色
```javascript
// 修改前
{ name: '待上报', type: 'bar', stack: 'status', data: [...], itemStyle: { color: '#fa8c16' } }

// 修改后
{ name: '待上报', type: 'bar', stack: 'status', data: [...], itemStyle: { color: '#fadb14' } }
```

#### 1.2 删除下方按钮区域
```vue
<!-- 删除以下代码块 -->
<div class="mt-3 flex justify-center gap-8">
  <button class="...应上报..." @click="handleTimelinessDrill('pending')">
    ...
  </button>
  <button class="...已上报..." @click="handleTimelinessDrill('reported')">
    ...
  </button>
</div>
```

### 第二步：修改4月营收偏差情况布局

#### 2.1 改为横向网格布局
```vue
<!-- 修改前：竖向列表 -->
<div class="space-y-2 max-h-80 overflow-y-auto">
  <div v-for="item in deviationData" :key="item.unit" class="...">
    <!-- 竖向卡片 -->
  </div>
</div>

<!-- 修改后：横向网格 -->
<div class="grid grid-cols-3 lg:grid-cols-5 gap-4">
  <div v-for="item in deviationData" :key="item.unit" class="bg-gray-50 rounded-lg p-3 hover:bg-gray-100 transition-colors cursor-pointer">
    <!-- 横向卡片 -->
  </div>
</div>
```

#### 2.2 调整卡片内容布局
每个公司的偏差信息横向展示：
- 公司名称（顶部居中）
- 项目偏差数、重点项目、一般项目（三列小网格）
- 偏差额（底部）

## 风险评估

- **风险等级**：低
- **主要风险**：横向布局在窄屏幕上可能拥挤
- **应对措施**：使用响应式网格（lg:grid-cols-5, 默认grid-cols-3）
