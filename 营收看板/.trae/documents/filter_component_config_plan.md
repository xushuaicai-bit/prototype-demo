# 页面顶部筛选组件配置计划

## 需求分析

根据用户需求，需要完成以下配置：

### 1. 筛选选项配置
- 一级筛选选项：全部、管网事业部、生态事业部、区域事业部、市政事业部、环境建设、管道工程、管道分公司、运营养护、二次养护
- 每个选项显示格式：选项名称(数量)
- 全部选项数量为所有基层单位项目数量之和

### 2. 营收看板交互功能
- 为7个指标卡片添加点击事件
- 所有可点击指标显示"点击查看详情"提示
- 点击后跳转至营收汇总表页面
- 营收汇总表同步筛选状态

### 3. 筛选状态保留
- 在营收看板与营收汇总表之间跳转时保留筛选条件
- 所有关联组件统一沿用筛选状态
