# 技术科研看板实现计划

## 1. 项目现状分析

当前项目是一个企业管理看板系统，包含以下主要看板：
- 供应链管理看板
- 生产看板
- 营收管理看板
- 市场管理看板

技术栈：Vue 3 + Vite + Tailwind CSS

## 2. 需求理解

根据用户提供的图片：
- **技术看板**：已有原型，包含审批完成率、方案回退率、交底完成率等统计指标，以及审批、方案、交底等详细图表，还有知识库使用情况
- **科研看板**：需要根据描述新建，包含科研项目/课题、科研成果情况、科研费用管控三大模块

要求：
- 将技术看板和科研看板合并为"技术科研看板"
- 布局合理美观
- 尽量在一个面板内显示，不翻页

## 3. 实现方案

### 3.1 文件结构

```
src/
├── views/
│   └── TechResearchDashboard.vue  # 技术科研看板主页面
├── components/
│   └── tech-research/
│       ├── TechStatCard.vue        # 技术统计卡片
│       ├── TechChart.vue           # 技术图表组件
│       ├── KnowledgeBase.vue       # 知识库组件
│       ├── ResearchProject.vue     # 科研项目组件
│       ├── ResearchAchievement.vue # 科研成果组件
│       └── ResearchExpense.vue     # 科研费用组件
└── data/
    └── mockData.js                 # 新增科研数据
```

### 3.2 组件设计

#### 3.2.1 技术看板部分
- 顶部导航：技术管理看板、科研管理看板、计量统计管理
- 标签页：全部、管网事业部、生态事业部等
- 统计卡片：总体审批完成率、总体方案回退率、总体交底完成率
- 图表区：审批完成率、方案回退率、交底情况
- 底部：知识库使用情况

#### 3.2.2 科研看板部分
- 科研项目/课题：
  - 重点项目进展情况（开题评审、中期评审、结题评审）
  - 按分公司/事业部跟踪进度，关键节点提醒
  - 数字+清单展示
- 科研成果情况：
  - 各类型成果统计（发明专利、实用新型、工法、标准、论文、软著、奖项、平台申报）
  - 每月新获得成果列表
  - 年度重点成果
  - 年度成果对比
- 科研费用管控：
  - 季度/年度科研费用情况
  - 进度条（目标值、实际值、完成率、偏差值）
  - 可按季度/年度筛选

### 3.3 布局规划

采用响应式网格布局：
- 第一行：标题区 + 筛选区
- 第二行：技术看板统计卡片 + 科研看板统计卡片
- 第三行：技术看板图表区 + 科研项目区
- 第四行：技术看板知识库 + 科研成果区
- 第五行：科研费用管控区

## 4. 实现步骤

1. **创建技术科研看板主页面** (`TechResearchDashboard.vue`)
2. **创建技术看板子组件**
3. **创建科研看板子组件**
4. **在 mockData.js 中添加模拟数据**
5. **在 Sidebar.vue 中添加"技术科研看板"菜单项**
6. **在 App.vue 中添加路由处理**
7. **测试和优化布局**
