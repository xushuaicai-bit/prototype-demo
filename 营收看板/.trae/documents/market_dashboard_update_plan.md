# 市场管理看板更新计划

## 需求分析

用户要求对市场管理看板进行以下修改：

1. **客户管理和客户排名上下排布**
   - 当前客户管理和客户排名是左右并排显示
   - 需要改为上下垂直排布
   - 重点客户数量值保持一致
   - 将"预估体量"、"履约体量"改为"跟踪体量"

2. **增加重点跟踪项目模块**
   - 创建新的柱状图模块
   - 横轴为9家基层单位
   - 纵轴为项目数量
   - 支持下钻到市场跟踪管理页面（筛选条件：是否为重点项目=是）

## 现有代码结构

```
src/views/
└── MarketDashboard.vue    # 市场管理看板主组件
```

## 修改方案

### 1. 修改客户管理和客户排名布局

| 修改内容 | 说明 |
|---------|------|
| 布局结构 | 将 lg:grid-cols-4 中的两个1列模块改为上下排布 |
| 标签文字 | 将"履约体量"改为"跟踪体量" |
| 底部说明文字 | 将"预估体量"改为"跟踪体量" |

### 2. 增加重点跟踪项目模块

| 内容 | 说明 |
|------|------|
| 模块标题 | 重点跟踪项目 |
| 横轴 | 9家基层单位（管网事业部、生态事业部、区域事业部、市政事业部、环境建设、管道工程、管道分公司、运营养护、二次养护） |
| 纵轴 | 项目数量 |
| 下钻功能 | 点击图表跳转至市场跟踪管理页面，筛选条件：是否为重点项目=是 |

### 3. 文件修改清单

| 文件 | 操作 | 说明 |
|------|------|------|
| `src/views/MarketDashboard.vue` | 修改 | 调整客户管理和客户排名布局 |
| `src/views/MarketDashboard.vue` | 修改 | 修改标签文字为跟踪体量 |
| `src/views/MarketDashboard.vue` | 新增 | 添加重点跟踪项目柱状图模块 |
| `src/App.vue` | 修改 | 添加下钻参数传递支持 |

## 技术要点

- 使用 ECharts 柱状图展示重点跟踪项目数据
- 使用 Element Plus 组件保持样式一致
- 实现图表点击下钻功能
- 响应式布局适配不同屏幕
