# 营收计划上报及时性和偏差情况调整计划

## 需求概述

### 1. 营收计划上报及时性调整
- **图例颜色修改**：浅黄色代表待上报（当前为橙色）
- **删除按钮**：移除图表下方的红色"应上报"和绿色"已上报"两个方块按钮

### 2. 4月营收偏差情况布局调整
- **当前问题**：竖向下拉显示，比例不协调
- **目标**：每个公司分模块横向显示（网格布局）

## 详细实施步骤

### 第一步：修改营收计划上报及时性

#### 1.1 修改"待上报"颜色为浅黄色
```javascript
// 修改前（橙色）
{ name: '待上报', type: 'bar', stack: 'status', data: [...], itemStyle: { color: '#fa8c16' } }

// 修改后（浅黄色）
{ name: '待上报', type: 'bar', stack: 'status', data: [...], itemStyle: { color: '#fadb14' } }
```

#### 1.2 删除下方按钮区域
删除第180-203行的按钮代码块（红色应上报和绿色已上报按钮）

### 第二步：修改4月营收偏差情况布局

#### 2.1 改为横向网格布局
```vue
<!-- 修改前：竖向列表 -->
<div class="space-y-2 max-h-80 overflow-y-auto">
  <div v-for="item in deviationData">...</div>
</div>

<!-- 修改后：横向网格 -->
<div class="grid grid-cols-3 lg:grid-cols-5 gap-3">
  <div v-for="item in deviationData" class="bg-gray-50 rounded-lg p-3 hover:bg-gray-100 cursor-pointer">
    <!-- 横向卡片 -->
  </div>
</div>
```

#### 2.2 卡片内容调整
每个公司卡片包含：
- 公司名称（顶部居中）
- 项目偏差数、重点项目、一般项目（三列小网格）
- 偏差额（底部右对齐）
