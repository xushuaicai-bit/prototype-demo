# 营收管理看板入口修复计划

## 问题分析

根据代码搜索结果：
- `standalone.html` 中包含了完整的**营收管理看板**原代码
- 当前 `Dashboard.vue` 显示的是"生产管理看板"，不是营收管理看板
- 左侧导航菜单中缺少"营收管理看板"入口
- `App.vue` 中没有对应的营收管理看板路由逻辑

## 修复方案

### 方案概述
基于 `standalone.html` 中的代码，恢复营收管理看板功能。

### 修改文件

1. **创建/恢复 src/views/RevenueDashboard.vue**

   - 包含：营收统计卡片、图表、筛选功能等

2. **修改 src/components/layout/Sidebar.vue**
   - 在左侧导航中添加"营收管理看板"菜单项
   - 将其放在显眼位置

3. **修改 src/App.vue**
   - 添加营收管理看板的视图路由逻辑
   - 处理顶部导航"产运管理"点击事件

4. **修改 src/data/mockData.js**
   - 确保营收管理看板所需数据可用

### 实施步骤

1. 读取 standalone.html 中的营收管理看板完整代码
